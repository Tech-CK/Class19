Introduction Html & Css

What is HTML and What is CSS ?

The Html stands for Hyper Text Markup Language and it is the standard markup language for creating Web pages. HyperText is effectively a text within a text.While CSS is the acronym for Cascade Styling Sheets. In short, it is a sheet style language, which is a type of language you can use to describe the presentation of a markup language ‐ in this case, to describe the movements of HTML. It effectively determines how the building blocks, as laid by HTML, are decorated and presented to the user.

Introduction Javascript

What is JS ?

JavaScript is a dynamic computer programming language. It is lightweight and most commonly used as a part of web pages, whose implementations allow client-side script to interact with the user and make dynamic pages. It is an interpreted programming language with object-oriented capabilities.
